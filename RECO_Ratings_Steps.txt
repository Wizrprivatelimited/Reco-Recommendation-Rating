RECO Ratings Code - All Steps Explained 

1. Open the course data file.

2. Read these two columns:
   - course_duration__c: Shows how long the course is (e.g., 5 or 2.5).
   - duration_medium__c: Shows the unit (e.g., "Days", "Months").

3. Fix the duration units for consistency:
   - "Day" → "Days"  
   - "Month" → "Months"  
   - "Year" → "Years"  

4. Define the number of seconds in each time unit:
   - 1 Year = 31,536,000 seconds  
   - 1 Month = 2,629,743 seconds (average)  
   - 1 Week = 604,800 seconds  
   - 1 Day = 86,400 seconds  
   - 1 Hour = 3,600 seconds  
   - 1 Minute = 60 seconds  

5. Convert all course durations into seconds.

6. Add a new column called Duration_in_seconds to store the converted values.

7. Add a column called Is_within_a_week:
   - If the course is exactly 7 days  mark it as True, else False.

8. Mark Non-Corporate (Wizr Courses) integrations:
   - Create an Integration column.
   - Set it to True if the provider is Guvi Geeks, Edujournal, Sharpener, or Business Toys Skills School. Otherwise, set to False.

9. Count number of concepts in each course:
   - Use the "concepts" column.
   - Count how many commas are there (each comma means another concept).

10. Handle missing concepts:
    - If the "concepts" column is empty:
      - Set concept_count = 0  
      - Set show = True  
      - Set concept = False  

11. Mark Priority_cpp for Non-Corporate courses((Wizr courses):
    - If the provider is Henry Harvin or Eduonix, mark Priority_cpp as True.

12. Mark Udemy courses:
    - If the provider is Udemy, mark Udemy_course as True.

13. Mark Coursera courses (Only for ABC data):
    - If the provider is Coursera, mark ABC_coursera as True.

14. Mark ABC courses:
    - If the provider is GVC, Gyanodaya Virtual Campus, or CIEL, mark ABC as True.

15. Rank the courses:
    - Use these columns for ranking:
      - Duration_in_seconds  
      - Integration  
      - INR amount

16. Save the final file with columns in this order:
    - Non-corporate Priority_cpp  
    - Non_corporate  
    - ABC Coursera  
    - ABC Courses  
    - Udemy Courses
